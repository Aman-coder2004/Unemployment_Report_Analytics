# Unemployment_Report_Analytics
📌 Project Overview
This project is an interactive Power BI Dashboard that analyzes unemployment trends in India during 2019 and 2020, with a major focus on the impact of the COVID-19 pandemic on employment across states, areas, and months.

The dashboard provides insights into:

State-wise unemployment trends
Urban vs Rural unemployment comparison
Month-wise unemployment fluctuations
Employment comparison between 2019 and 2020

🎯 Problem Statement
The COVID-19 outbreak in 2020 severely affected the Indian economy.
Lockdowns, business shutdowns, migration crises, and reduced industrial activity led to a rapid increase in unemployment across the country.

This project aims to analyze:

📉 The rise in unemployment after COVID-19
🏙️ States most affected by unemployment
🌆 Area-wise unemployment (Urban vs Rural)
📅 Monthly unemployment trends during the pandemic
👨‍💼 Comparison of employed population in 2019 vs 2020

The dashboard helps users understand how employment patterns changed during the pandemic using visual analytics.
